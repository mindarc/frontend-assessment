Explain why the result of `('b' + 'a' + + 'a' + 'a').toLowerCase()` is `banana`.

The '+' is used to concatenate strings when it is inserted between 2 strings.
Result: 'b' + 'a' = ba

If it is placed next to a string, it converts the string to an integer. If the string is not a valid number, it returns NaN (Not A Number)
Result: 'b' + 'a' + (+'a') = baNaN

When concatenated all together.
Result: 'b' + 'a' + (+ 'a') + 'a' = baNaNa

The method toLowerCase() converts the string to be displayed in small letters.
Result: ('b' + 'a' + + 'a' + 'a').toLowerCase() = banana
